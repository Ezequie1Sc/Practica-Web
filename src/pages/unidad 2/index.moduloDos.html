¡Bienvenido al Módulo 2!

El objetivo de este módulo es que los estudiantes repasen conceptos básicos de programación que han aprendido durante su formación académica y que son indispensables para los siguientes módulos del Programa Células Plenum.

Duración: 12 días 7 temas
Comenzar Módulo

Temas del Módulo

2.1 Fundamentos de Bases de Datos

Conoce los conceptos esenciales de las bases de datos, su estructura y funcionamiento básico.

Comenzar Tema

2.2 Normalización de Bases de Datos

Aprende a organizar los datos de manera eficiente para evitar redundancias y anomalías.

Comenzar Tema

2.3 Lenguaje de Consulta Estructurado (SQL)

Domina el lenguaje estándar para gestionar y consultar bases de datos relacionales.

Comenzar Tema

2.4 Triggers en Bases de Datos

Descubre cómo automatizar acciones en respuesta a eventos específicos en la base de datos.

Comenzar Tema

2.5 Paradigmas de Programación

Explora los diferentes enfoques para resolver problemas mediante la programación.

Comenzar Tema

2.6 Java y Programación Orientada a Objetos

Aprende los fundamentos de Java y los principios de la programación orientada a objetos.

Comenzar Tema

2.7 Arquitectura de Software y APIs

Comprende cómo diseñar sistemas escalables y el funcionamiento de las interfaces de programación.

Comenzar Tema
